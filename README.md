#### Installing 
Don't forget to add this line ***glory_convention_lint: ^x.x.x*** to your pubspec.yaml file, and run ***flutter pub get*** from terminal.
~~~gradle
dev_dependencies:
  glory_convention_lint: ^0.0.1
~~~
